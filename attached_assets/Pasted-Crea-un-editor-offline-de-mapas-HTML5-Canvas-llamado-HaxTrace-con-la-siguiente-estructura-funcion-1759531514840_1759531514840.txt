Crea un editor offline de mapas HTML5 + Canvas llamado HaxTrace con la siguiente estructura, funcionalidades y estilo visual inspirado en las imágenes proporcionadas: 1. Estructura y layout Header / Main Menu Logo “HaxTrace” a la izquierda. Enlaces: Home, Editor, Import/Export, Settings, Help. Botón de usuario o ajustes rápidos a la derecha. Sidebar izquierda (debajo del header) Sección “Map Settings”: Nombre del mapa (input). Ancho y alto (inputs numéricos). Sección “Background”: Selector de color e imagen (preview). Sliders de opacidad y escala. Toolbar horizontal (sobre el canvas) Iconos de herramientas: Add Vertex Add Segment Pan Zoom In / Zoom Out Undo / Redo Canvas central Fondo oscuro con grid sutil (#2e2e42 sobre #1e1e2e). Vértices y segmentos con acentos neón (cyan #00e5ff). Sidebar derecha Panel “Vertex List”: lista tabular de vértices (ID, X, Y). Panel “Segment List”: lista de segmentos (start→end, curve). Panel de propiedades: editar curva (slider + input), color. Footer / Status Bar Herramienta activa, nivel de zoom, coordenadas del mouse. 2. Estilo visual (CSS) :root { –bg: #1e1e2e; –grid: #2e2e42; –accent: #00e5ff; –text: #e0e0e0; –hover-glow: drop-shadow(0 0 8px var(--accent)); } Body y contenedores: fondo var(--bg), tipografía sans-serif (Poppins/Montserrat). Header y botones: texto uppercase, hover con glow var(--hover-glow). Canvas: grid background via CSS o dibujado en renderer.js.. Sidebars: paneles colapsables con borde highlight var(--accent). Sliders e inputs: track customizado en var(--accent). 3. Módulos y archivos (esqueleto) index.html Header, sidebars, toolbar, canvas, footer. Import de styles.css y main.js.. styles.css Variables de tema y layout grid/flex. Clases para header, sidebar, toolbar, canvas-container, footer. mapData.js Exporta objeto con { name, width, height, vertices: [], segments: [] }. Métodos: addVertex, addSegment, moveVertex, updateCurve. renderer.js Funciones: drawGrid, drawBackground, drawVertices, drawSegments. getCurvePoints para renderizar arcos. tools.js Control de estado de herramienta activa. Handlers: onCanvasClick, onCanvasDrag. uiManager.js Actualiza listas en sidebars y sliders. Sincroniza inputs con mapData. importExport.js Export .hbs (JSON) y descarga como ZIP opcional. Import y parse JSON para rellenar mapData. main.js Inicialización: crear canvas, listeners, history stack para undo/redo. Prevención de menú contextual para clic derecho. 4. Comportamiento esperado Añadir vértice: clic izquierdo en canvas → mapData.addVertex + renderer. Crear segmento: seleccionar herramienta Segment → clic en dos vértices. Mover vértice: arrastrar con clic derecho. Editar curva: slider en panel o arrastrar segmento horizontalmente. Zoom/Pan: rueda de ratón para zoom, arrastrar canvas para pan. Undo/Redo: snapshot en main.js cada acción. Import/Export: botones en header o menú lateral. hazlo con estilo moderno, no cyberpunk
