HaxTrace — breve explicación para Replit (estilo GitHub)
HaxTrace no trabaja a nivel de píxeles sino con un sistema de vértices y segmentos pensando en mapas (tipo Haxball). A continuación se explica de forma clara qué significan los datos y cómo exportar/importar correctamente.

Concepto clave
En vez de editar píxeles, construyes el mapa con:
Vértices (vertex): puntos con coordenadas world (x, y).
Segmentos (segments): conexiones entre dos vértices (referenciadas por índices), pueden ser rectas o curvas y pueden llevar color.
El canvas es solo la vista: las posiciones que ves se traducen a coordenadas del mapa (no coordenadas CSS/píxel absolutas).
Estructura de datos (resumen)
Un mapa es un JSON con:

name, width, height
bg: { color }
vertexes: [ { x, y }, ... ]
segments: [ { v0: index, v1: index, color?: "hex", curve?: number }, ... ]
Ejemplo (simplificado):
{
  "name":"New Stadium",
  "vertexes":[{"x":-236,"y":-111},{"x":-86,"y":-112}],
  "segments":[{"v0":0,"v1":1,"color":"000000","curve":30}]
}

Curvas: nota importante de compatibilidad (.hbs / Haxball)
El editor guarda curve con la convención interna, pero al exportar/importar se invierte el signo para ser compatible con Haxball.
Export: el valor curve se multiplica por -1 antes de descargar.
Import: la curva del archivo importado se multiplica por -1 al cargar.
Resultado: usa el botón Export HBS para obtener un .hbs listo para Haxball.
Herramientas principales (cómo usarlas)
Add Vertex: clic izquierdo en canvas → crea un vértice en la posición world.
Add Segment: selecciona dos vértices (clic sobre vértice) → crea segmento entre ellos con color/curve desde la sidebar.
Pan: arrastra para desplazar la vista.
Mover vértice: botón derecho sobre un vértice → arrastra para reposicionarlo.
Editar curva: selecciona un segmento → usa panel (slider/número) o arrastra horizontalmente con botón derecho para ajustar curve.
Undo/Redo: Ctrl+Z / Ctrl+Y (historial de snapshots).
Export / Import
Export HBS: serializa el mapa y descarga .hbs (JSON) con curvas invertidas para Haxball.
Import HBS: selecciona archivo .hbs/.json. Al importar las curvas se invierten para adaptarse al editor.
Consejos para usar en Replit
Abre index.html en Replit (Live preview). Interactúa desde la sidebar.
Cuando termines, usa Export HBS para descargar el .hbs y súbelo a Haxball o compártelo.
Si compartes el proyecto en GitHub, agrega este README para orientar a otros sobre la convención de curvas (la inversión de signo es la trampa más común).
